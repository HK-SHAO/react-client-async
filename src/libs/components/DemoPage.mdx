## ğŸ¬ `Demo` of Recursive Async Component

Easy to `create` an memoized async component with recursion.

```tsx
const Rec = create<{ n: number }>(
  async ({ [$signal]: signal, n }) =>
    // break the recursion
    (n <= 0) ? 0 : (
    // delay and recursion
      <>
        {await delay(99, signal)}
        {n} <Rec n={n - 1} /> {n}
      </>
    )
);
```
