## <a href='#RecursiveDemo' id="RecursiveDemo">ğŸ¬ `Demo` of Recursive Async Component</a>

Easy to `wrap` a recursive async component and memoize it.

```tsx
const Rec = wrap<{ n: number }>(
  async ({ [$signal]: signal, n }) =>
    // break the recursion
    (n <= 0) ? 0 : (
    // delay and recursion
      <>
        {await delay(99, signal)}
        {n} <Rec n={n - 1} /> {n}
      </>
    )
);
```
